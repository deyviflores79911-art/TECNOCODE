# Guía para Ejecutar la Aplicación

Para ejecutar esta aplicación de ventas de celulares, sigue estos pasos:

### 1. Instalar el SDK de Flutter
Si aún no lo tienes, descárgalo e instálalo:
- **Descargar**: [Flutter SDK para Windows](https://docs.flutter.dev/get-started/install/windows)
- **Extraer**: Descomprime el archivo en una carpeta (ej: `C:\src\flutter`).
- **PATH**: Añade `C:\src\flutter\bin` a las variables de entorno de tu sistema.

### 2. Configurar el Editor (Recomendado)
- Usa **VS Code**.
- Instala la extensión oficial de **Flutter** (esto instalará automáticamente la de Dart).

### 3. Instalar Dependencias
Abre una terminal en la carpeta del proyecto (`c:\Users\msi\Desktop\flutter`) y ejecuta:
```powershell
flutter pub get
```

### 4. Lanzar la Aplicación
Conecta un celular por USB o abre un navegador (Chrome) y ejecuta:
```powershell
flutter run
```
*Si usas VS Code, simplemente presiona **F5**.*
